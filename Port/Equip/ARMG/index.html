Automated Rail-Mounted Gantry

Hoisting capacity 41 t · Rail span 40 m · Hoisting height 18.2 m

Operation Cycle

  1. Crane traverses to the waiting AGV.
  2. Spreader lowers, clamps the container, then hoists.
  3. Trolley shifts to the yard block.
  4. Container is centered over the stack and lowered.
  5. Spreader releases, rises, and returns to standby.
